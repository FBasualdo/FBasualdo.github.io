Proyectos

Pass Guardian

Pass Guardian fue un proyecto que nació pensando en mis padres y su dificultad a la hora de recordar contraseñas, e-mails y usuarios.

Consta de una aplicacion de escritorio que posee dos apartados:
- Generador de contraseñas donde puedes escoger entre ciertos parametros como habilitar caracteres especiales, activar mayusculas y minusculas, y que se genere palabras con sentido.
- Gestor de cuentas conectada a una base de datos donde para cada cuenta se almacena Referencia (Instagram, Banco), User (e-mail, nombre de usuario) y Contraseña

Desarrollado con: Python, SQLite

AutoTrend

AutoTrend es mi primer gran proyecto. Surgió a partir del lanzamiendo de ChatGPT y el sentimiendo de ver cuanto puede ser explotada esta tecnología.

Consta de una programa basado en consola al cual le ingresas uno o mas prompts del estilo: "Cómo preparar una pizza?", o "Tres curiosidades sobre el imperio romano", y un video de fondo.
El programa se encarga de ingresar el prompt a ChatGPT, crear subtitulos en base a la respuesta, utilizar un TTE (Text to speech), grabar la voz y recortar el video automaticamente.

Una vez realizado tienes el video listo para subir en cualquier plataforma.
Proximas Acutalizaciones: * Lograr que el programa suba los videos automaticamente a Youtube y Tiktok

Desarrollado con: Python
